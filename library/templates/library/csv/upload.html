CSV処理

CSVをアップロードし、データを操作します。下のタブからモデルの種類を選択してください。
  • 事業者
  • 路線
  • 駅

1. CSVファイルをダウンロードしてください

データベースに格納されているデータを CSV ファイル形式でダウンロードします。

CSVファイルダウンロード

2. CSVファイルを編集してください

ヘッダー行は削除しないでください。

A列. command

command 列の内容を "download" から変更することで、処理方法を下記の中から選択することができます。

command 説明
add 当該行の内容を追加します。
download 何もしません。

B列. id

オブジェクトの id を uuid 形式で入力します。

C列以降

オブジェクトの情報を入力してください。

3. CSVファイルをアップロードしてください

下記のフォームに CSV ファイルをアップロードし「確認」を選択すると処理を開始します。
